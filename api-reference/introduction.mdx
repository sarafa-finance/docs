## Authentication

All API endpoints are authenticated using API keys.

```json
"headers":
  {
    "api_key": ""
  }
```
## Base URL

All API endpoints are hosted on the following base URL:
For Development:
```
https://dev.api.sarafa.io/api/
```
For Production:
```
https://api.sarafa.io/api/
```

## Rate Limiting
All API requests are limited to 5 Requests per second. If your application requires more,
please email us at info@sarafa.io
